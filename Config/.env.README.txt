────────────────────────────────────────────────────────────────
PURPOSE
────────────────────────────────────────────────────────────────
Configures TensorFlow environment variables for:
  • Suppressing TensorFlow warning messages
  • Enabling CPU optimizations (oneDNN)
  • Ensuring clean terminal output

────────────────────────────────────────────────────────────────
CONTENTS
────────────────────────────────────────────────────────────────
export TF_CPP_MIN_LOG_LEVEL=2     # Suppress info/debug (2=warnings+errors)
export TF_ENABLE_ONEDNN_OPTS=1    # Enable Intel oneDNN optimizations
